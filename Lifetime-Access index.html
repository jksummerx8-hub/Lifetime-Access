更換裝置或重新安裝？

本產品權限與您的 Apple ID 永久綁定。若您更換 iPhone、iPad 或重新安裝 App，只需登入同一個 Apple ID 並點擊頁面下方的「恢復購買」，即可立即找回所有已解鎖的功能，無需重複付費。

一次購買，跨裝置終身通行

我們深知 Zenith 對您的陪伴意義重大。選擇「終身版」方案，您的專業權限將永久記錄在 Apple 雲端憑證中。無論未來您更換了哪一代 iPhone，只要在設定中點擊「恢復購買」，您的沉浸空間與專屬功能都將無縫回歸。這是我們對您支持的長久承諾。

Q：如果我買了終身版後換手機，需要重新付費嗎？

A：完全不需要。

帳號綁定： 您的購買紀錄是儲存在 Apple 的加密伺服器中，與您的 Apple ID 連結。
跨裝置使用： 您可以在所有登入相同 Apple ID 的 iOS 裝置上共用此權限。
操作簡單： 在新手機上進入購買頁面，點擊底部的 「恢復購買 (Restore Purchase)」，系統驗證成功後即可自動解鎖。
